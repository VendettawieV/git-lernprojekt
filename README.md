# 🐙 Git-Lernprojekt

Willkommen! Dieses Repository ist dein Übungsplatz, um Git zu lernen.
Alle Aufgaben findest du im Ordner `aufgaben/`.

## 🚀 Erste Schritte – So klonst du das Repo

```bash
git clone https://github.com/VendettawieV/git-lernprojekt.git
cd git-lernprojekt
```

## 📋 Übersicht der Aufgaben

| Übung | Thema                    | Branch                        |
|-------|--------------------------|-------------------------------|
| 01    | Erster Commit            | `uebung/01-erster-commit`     |
| 02    | Branches & Commits       | `uebung/02-branching`         |
| 03    | Merge Conflict lösen     | `uebung/03-merge-conflict`    |
| 04    | Fehler rückgängig machen | `uebung/04-revert`            |
| 05    | Git Log & History        | `uebung/05-git-log`           |

## 📖 So startest du eine Aufgabe

```bash
# Branch wechseln (z.B. Aufgabe 1):
git checkout uebung/01-erster-commit

# Aufgabe lesen:
cat aufgaben/uebung-01.md
```

Viel Spaß! 💪
